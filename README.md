# Environment setup

## Python 
install python version 3.5 from:
https://www.python.org/downloads/release/python-353/

## Required packages
- scikit learn: 
**`pip install sklearn`**

- numpy, scipy:
http://www.lfd.uci.edu/~gohlke/pythonlibs/

- tensor flow:
**`pip install tensorflow`**

- tflearn:
**`pip install tflearn`**
